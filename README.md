# server

Clone the repo to local

```bash
git clone https://github.com/forhadreza43/role-based-better-auth-postgresql-drizzle-backend.git
```

Install dependencies:

```bash
bun install
```

To run:

```bash
bun run dev
```

To update schema

```bash
bun run db:generate
```

To sync new schema to database

```bash
bun run db:migrate
```
